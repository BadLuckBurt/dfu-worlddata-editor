Unity RMB Data Merger

Load RMB JSON

Please select the RMB JSON file you exported from Daggerfall Unity and click the Load JSON button

Explanation

Click the label or the checkbox of the RMB properties you want merge and choose the correct file for the upload field that gets enabled.

When you are ready, click the Merge button and copy the new JSON contents from the Results textarea and paste them into your RMB.json file.
